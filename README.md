## Getting Started

To get started with my dotFiles, simply download the source code and run as a flake.

### Prerequisites

- Nixos
- Flakes

### Installing

1. Clone the repository  
   ``` git clone https://github.com/magicalzfmk/nixosConfigs.git ```
2. Open the project in terminal
3. run as a flake
